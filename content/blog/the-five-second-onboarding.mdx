## Tours treat the symptom

Multi-step product tours exist because the first screen failed to explain
itself. Users skip them, then meet the same confusing screen — now with no
help at all.

## Design the first five seconds

Show real value on the first screen, with the user's own words on the
buttons. One honest empty state beats five tooltips every time.
